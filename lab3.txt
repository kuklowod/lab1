Нормальное состояние системы

	В системе существуют пользователи который могут выполнять локальный и удаленный вход в систему:
		- root
		- user0
		- student
	Пароли пользователей должны быть не короче 7 символов.

	В системе существуют следующие сетевые сервисы:
		- ssh
Известно что система была взломана злоумышлеником.

Цель лабораторной работы:
	Найти изменения произведенные злоумышлеником, оценить их последствия

Всего злоумышленником внесено примерно 6 изменений, направленных на локальное и удаленное получение
доступа к хосту.

В директории /home/backups находятся файлы с листингом ls -lR / и md5 суммой всех файлов до модификации
злоумышленником.

В отчет поместить описание изменений; объяснение целей, с которыми они были добавлены. В выводы: собственное
мнение о том, что необходимо делать в подобной ситуации в повседневной жизни

Для выполнения лабораторной работы необходимо и достаточно знаний полученных на лабораторной работе №2

Минимальный набор утилит необходимых для выполнения лабораторной работы:
	diff - утилита сравнивает два файла и выводит на экран строки которые отличаются в этих файлах
	md5sum - утилита для подсчета md5 контрольной суммы
	find - утилита для поиска файлов на файловой системе
	grep - выполняет поиск заданой строки в файле
	ls - утилита для отображения содержимого каталога
	netstat - утилита для просмотра активных сетевых соединений хоста
	getent - утилита для универсального запроса nss
	mc - файловый менеджер
	last - утилита предназначена для вывода списка пользователей недавно входивших в систему
	who - список пользователей находящихся в системе
	id - утилита выводит текущий uid и gid пользователя
	xargs - утилита для построения списка аргументов выполняемой команды
